Получить Resonance Amulet можно двумя способами. Один способ -победить *Archon of Halisha*, который находится внутри *FourSepulchers*. Тебе нужно будет собрать партию, чтобы войти. Если этоокажется слишком сложно, победи последователей Halisha в *Shrine ofLoyalty*. Сделав так, ты можешь заработать "билеты". Как только наберешьбольше *700* билетов, появится сам Halisha. Между прочим, создания,которых тебе нужно победить, это *Grave Scarab, Scavenger Scarab, GraveAnt, Scavenger Ant, Shrine Knight и Stander*.
