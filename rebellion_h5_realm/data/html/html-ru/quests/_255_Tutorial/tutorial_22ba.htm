﻿ [Изменение класса Разбойника]:

Чтобы стать Искателем Сокровищ, Разбойник должен успешно пройти три квеста. Испытание Искателя может проходить только игрок, достигший уровня 35 и выше. Этот квест начинается с визита к Мастеру Дафнеру из Города Гиран. Испытание Доверия может проходить только игрок, достигший уровня 37 и выше. Этот квест начинается со встречи с Верховным Жрецом Холлинтом из Города Орен. Испытание Искателя может проходить только игрок, достигший уровня 39 и выше. Этот квест начинается со встречи с Мастером Лютером из Деревни Охотников .
