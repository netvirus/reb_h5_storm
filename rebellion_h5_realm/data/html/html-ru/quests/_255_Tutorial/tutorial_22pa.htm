﻿ [Изменение класса Темного Волшебника]:

Чтобы стать Заклинателем Ветра, игрок должен успешно пройти три квеста. Испытание Мудрости может проходить только игрок, достигший уровня 35 и выше. Этот квест начинается со встречи с Магистром Мириен из Города Дион. Испытание Судьбы может проходить только игрок, достигший уровня 37 и выше. Этот квест начинается со встречи с Магистром Кайрой из Города Гиран. Наконец, Испытание Мага может проходить только игрок, достигший уровня 39 и выше. Этот квест начинается со встречи с Бардом Рукалом из Города Дион.
