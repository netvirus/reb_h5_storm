﻿ [Изменение класса Светлого Волшебника]:

Чтобы стать Светлым магом, персонаж должен успешно пройти три квеста. Испытание Мудрости может проходить только игрок, достигший уровня 35 и выше. Этот квест начинается со встречи с Магистром Мириен из Города Дион. Испытание Жизни может проходить только игрок, достигший уровня 37 и выше. Этот квест начинается со встречи с Мастером Кардиеном из Города Дион. Наконец, Испытание Мага может проходить только игрок, достигший уровня 39 и выше. Этот квест начинается со встречи с Бардом Рукалом из Города Дион.
