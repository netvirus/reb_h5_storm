Чтобы стать Мстителем, вы должны успешно пройти три квеста. Испытание Верности может проходить только игрок, достигший уровня 35. Этот квест начинается с визита к Великому Мастеру Ханнавальту из Города Гиран . Испытание Доверия может проходить только игрок, достигший уровня 37. Этот квест начинает Верховный Жрец Холлинт из Города Орен. Испытание Колдовства может проходить только игрок, достигший уровня 39. Этот квест начинается с посещения Тени Орима у Академии Хардина.
