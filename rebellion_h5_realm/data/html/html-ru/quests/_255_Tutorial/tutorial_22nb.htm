﻿ [Изменение класса Темного Рыцаря]:

Чтобы стать Темным Танцором, игрок должен успешно пройти три квеста. Испытание Храбрости может проходить только игрок, достигший уровня 35 и выше. Этот квест начинается со встречи с Кэшем из Города Дион. Испытание Судьбы может проходить только игрок, достигший уровня уровня 37 и выше. Этот квест начинается со встречи с Магистром Кайрой из Города Гиран. Наконец, Испытание Чемпиона может проходить только игрок, достигший уровня 39 и выше. Этот квест начинается со встречи с Чемпионом Каеном из Города Орен.
