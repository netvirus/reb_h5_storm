﻿ [Изменение класса Рыцаря]:

Рыцарь, достигший уровня 40, может во второй раз изменить класс и стать Паладином или Мстителем. Чтобы изменить класс во второй раз, игрок должен успешно пройти три квеста. Первый из этих квестов могут проходить только персонажи, достигшие уровня 35. Если вы хотите узнать больше, выберите заинтересовавший вас класс.
