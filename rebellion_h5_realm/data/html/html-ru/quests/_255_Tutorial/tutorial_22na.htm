﻿ [Изменение класса Темного Рыцаря]:

Чтобы стать Рыцарем Шилен, игрок должен успешно пройти три квеста. Испытание Верности может проходить только игрок, достигший уровня уровня 35 и выше. Этот квест начинается со встречи с Великим Мастером Ханнавальтом Города Гиран. Испытание Судьбы может проходить только игрок, достигший уровня уровня 37 и выше. Этот квест начинается со встречи с Магистром Кайрой из Города Гиран. Наконец, Испытание Колдовства может проходить только игрок, достигший уровня уровня 39 и выше. Этот квест начинается со встречи с Тенью Орима у Академии Хардина.
