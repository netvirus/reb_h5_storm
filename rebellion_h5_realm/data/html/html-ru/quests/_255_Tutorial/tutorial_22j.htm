﻿ [Изменение класса Монаха]:

Монах, достигший уровня 40, может во второй раз изменить класс и стать Тираном. Чтобы стать Тираном, игрок должен успешно пройти три квеста. Испытание Храбрости может проходить только игрок, достигший уровня 35 и выше. Этот квест начинается со встречи с Кэшем из Города Дион. Испытание Славы может проходить только игрок, достигший уровня 37 и выше. Этот квест начинается со встречи с Префектом Вокианом из Города Гиран. Наконец, Испытание Чемпиона может проходить только игрок, достигший уровня 39 и выше. Этот квест начинается со встречи с Чемпионом Каеном из Города Орен.
