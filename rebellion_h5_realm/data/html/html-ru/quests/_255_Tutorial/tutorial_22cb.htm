Чтобы стать Некромантом, персонаж должен успешно пройти три квеста. Испытание Мудрости может проходить только игрок, достигший уровня 35 и выше. Этот квест начинается со встречи с Магистром Мириен из Города Дион. Испытание Доверия может проходить только игрок, достигший уровня 37. Этот квест начинается со встречи с Верховным Жрецом Холлинтом из Города Орен. Наконец, Испытание Колдовства может проходить только игрок, достигший уровня 39. Этот квест начинается со встречи с Тенью Орима у Академии Хардина.
