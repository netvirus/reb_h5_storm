﻿ [Изменение класса Разведчика]:

Чтобы стать Охотником, Разведчик должен успешно пройти три квеста. Испытание Искателя может проходить только игрок, достигший уровня 35 и выше. Этот квест начинается со встречи с Мастером Дафнером из Города Гиран. Испытание Жизни может проходить только игрок уровня 37 и выше. Этот квест начинается со встречи с Мастером Кардиеном из Города Дион. Наконец, Испытание Стрельца может проходить только игрок, достигший уровня 39 и выше. Этот квест начинается со встречи с Главой Гильдии Охотников Бернардом в Деревне Охотников.
