﻿ [Изменение класса Светлого Рыцаря]:

Чтобы стать Менестрелем, Светлый Рыцарь должен успешно пройти три квеста. Испытание Храбрости может проходить только игрок, достигший уровня 35 и выше. Этот квест начинается со встречи с Кэшем из Города Дион. Испытание Жизни может проходить только игрок, достигший уровня 37 и выше. Этот квест начинается со встречи с Мастером Кардиеном из Города Дион. Наконец, Испытание Чемпиона может проходить только игрок, достигший уровня 39 и выше. Этот квест начинается со встречи с Чемпионом Каеном из Города Орен.
