﻿ [Изменение класса Воителя]:

Чтобы стать Копейщиком, игрок должен успешно пройти три квеста. Испытание Храбрости может проходить только игрок, достигший уровня 35 и выше. Этот квест начинается со встречи с Кэшем из Города Дион. Испытание Доверия может проходить только игрок, достигший уровня 37 и выше. Этот квест начинается со встречи с Верховным Жрецом Холлинтом из Города Орен. Наконец, Испытание Чемпиона может проходить только игрок, достигший уровня 39 и выше. Этот квест начинается со встречи с Ветераном Аскалоном из Города Гиран.
