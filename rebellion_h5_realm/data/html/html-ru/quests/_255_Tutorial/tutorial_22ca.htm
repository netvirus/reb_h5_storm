﻿ [Изменение класса Чародея]:

Чтобы стать Волшебником или Волшебницей, персонаж должен успешно пройти три квеста. Испытание Мудрости может проходить только игрок, достигший уровня 35 и выше. Этот квест начинается со встречи с Магистром Мириен из Города Дион. Испытание Доверия может проходить только игрок, достигший уровня 37 и выше. Этот квест начинается со встречи с Верховным Жрецом Холлинтом из Города Орен. Испытание Мага может проходить только игрок, достигший уровня 39 и выше. Этот квест начинается со встречи с Бардом Рукалом из Города Дион.
