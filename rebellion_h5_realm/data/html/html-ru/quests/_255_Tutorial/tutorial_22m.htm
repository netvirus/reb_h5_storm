﻿ [Изменение класса Ремесленника]:

Ремесленник, достигший уровня 40, может во второй раз изменить класс и стать Кузнецом. Чтобы стать Кузнецом, игрок должен успешно пройти три квеста. Испытание Умения может проходить только игрок, достигший уровня 35 и выше. Этот квест начинается со встречи со Смотрителем Склада Вальконом из Города Гиран. Испытание Успеха может проходить только игрок, достигший уровня 37 и выше. Этот квест начинается со встречи со Смотрителем Склада Парманом из Города Гиран. Наконец, Испытание Мастера может проходить только игрок, достигший уровня 39 и выше. Этот квест начинается со встречи с Локрином из Гильдии Железных Врат в Деревне Гномов.
