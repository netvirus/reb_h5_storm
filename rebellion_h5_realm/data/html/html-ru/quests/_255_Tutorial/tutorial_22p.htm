﻿ [ Смена профессии Темного Волшебника ]:

Достигнув 40-го уровня Темный Волшебник может сменить профессию на Заклинатель Ветра или Спектрала. Для того, чтобы сменить 2-ю профессию, вам надо выполнить 3 квеста и первый квест необходимо выполнить на 35-м уровне. Для более подробной информации кликните на Классе.
