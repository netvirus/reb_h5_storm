Subclass Skill Certification Once you reach a certain level of skill inyour subclass, you skills can be certified. As a result, you will beable to learn some special skills which you can use in your main class. However, each subclass slot can have only *one certification*. Thatmeans that even if you come back with a different occupation for thesubclass, you cannot obtain another certification unless you first spenda *huge amount to cancel the first certification*. If you want to obtaina certification, *your current occupation should be that of the subclassyou want to be certified, and you should be above level 65*.
